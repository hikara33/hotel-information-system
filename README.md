# hotel-information-system

Курсовой проект по дисциплине "Алгоритмы и структуры данных":
информационная система "Регистрация постояльцев в гостинице".

## Используемые структуры и алгоритмы

- Хеш-таблица для постояльцев (ключ: номер паспорта).
- AVL-дерево для гостиничных номеров (ключ: номер комнаты).
- Слоеный список (skip list) для регистраций заселения/выселения, упорядоченный по номеру комнаты.
- Поиск подстроки Бойера-Мура:
  - для поиска постояльцев по ФИО;
  - для поиска номеров по фрагменту оборудования при прямом обходе AVL-дерева.
- Сортировка распределением (radix sort строк) для вывода списков гостей и комнат.

## Запуск

```bash
cd backend
npm install
npm run dev
```

Приложение доступно по адресу: `http://localhost:3000`.
